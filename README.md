# Cafe Order Queue

This is a simple Python program that simulates a cafe order queue using a **FIFO (First-In, First-Out)** system. Orders are stored in a queue, and the program serves them in the order they were received.

## Features

- Add orders to the queue.
- Serve orders in the order they were added.
- Print each order as it is served.
- Notify when all orders are completed.
